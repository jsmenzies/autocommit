This is a CLI tool that analyzes Git history and generates conventional commit messages using LLM providers. Written in Zig.

## Build Commands

- Build: `zig build`
- Test: `zig build test`
- Run: `zig build run`

## Commit Style

Use conventional commits: `<type>(<scope>): <subject>`
Types: feat, fix, docs, style, refactor, test, chore
